# SmoulderingEngine
This is a renderer I am working on in my free time. The end goal of this project is to make it into a small yet functional game engine that can be used to make 3D games. Currently the rendering system is influenced heavily by Sascha Willems (https://github.com/SaschaWillems), as I learn more about Vulkan my plan is to customize the rendering logic and implement memory/time optimizations.
